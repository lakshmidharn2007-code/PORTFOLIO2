WORK / 01

Telecom Customer Churn Prediction 2026

PROBLEM

Telecom churn is expensive and easy to miss until it's too late — raw model probabilities aren't actionable for non-technical business users.

SOLUTION

A Streamlit app that predicts stay/leave, explains the top reasons in plain language, and recommends specific retention strategies.

Model selection across Logistic Regression, Random Forest, and Gradient Boosting (best ROC-AUC wins), with a rule-based explanation engine and a live risk-probability gauge — built so a business user sees "Will Stay" or "Will Leave," not a raw score.

HOMIS — Hospital Management System 2026

PROBLEM

Hospitals need strict access control across roles (patient, nurse, doctor, pharmacy, reception) without five disconnected systems.

SOLUTION

One shared backend, one database, five role-scoped portals — each seeing only what its role is permitted to access.

A production-oriented monorepo built around JWT-based RBAC, auditable workflows, pharmacy fulfilment, secure payment orchestration, and signed-URL medical file access — full TypeScript coverage across a Turborepo workspace.

WORK / 04

Voice Command Model 2026

PROBLEM

Voice interfaces usually need one hardcoded intent per destination — brittle, and it doesn't scale to new sites.

SOLUTION

A generic "open website" intent that generalises to unseen site names, plus app-navigation intents, served from a single deployment.

Browser-based speech-to-text via the Web Speech API (audio never leaves the device), a custom-trained TF-IDF + Logistic Regression intent classifier through FastAPI, and a confidence threshold that rejects low-confidence guesses instead of acting on them.
